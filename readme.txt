== Description ==

Provide an easy to use api for post, term and comment reactions.

- Like
- Dislike
- Report
- ...

Reactions can be whatever you want.

== Installation ==

1. Upload `reaction.zip` to the `/wp-content/plugins/` directory
1. Extract the Plugin to a `reaction` Folder
1. Activate the plugin through the 'Plugins' menu in WordPress
